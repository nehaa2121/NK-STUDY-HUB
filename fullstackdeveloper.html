100 Days Full Stack + AI Roadmap

Phase 1 (Days 1–10): HTML & CSS (Frontend Basics)

Goal: Learn structure & styling of web pages

Topics:
  • HTML5: headings, paragraphs, links, images, lists, forms, tables
  • CSS3: selectors, box model, colors, borders, margin/padding
  • Flexbox & Grid
  • Responsive Design (media queries)
  • Mini Projects:
    • Personal portfolio layout
    • Simple landing page

Tools: VS Code, Chrome DevTools, GitHub



Phase 2 (Days 11–20): Advanced CSS & JavaScript Basics

Goal: Make websites interactive

Topics:
  • CSS animations, transitions
  • JavaScript syntax, variables, operators, data types
  • Functions, arrays, objects, loops, conditionals
  • DOM manipulation (getElementById, event listeners)
  • Mini Project:
    • To-do List App
    • Calculator

Phase 3 (Days 21–30): Advanced JavaScript + ES6

Goal: Deepen JS understanding

Topics:
  • Arrow functions, destructuring, spread/rest
  • Promises, async/await
  • Fetch API (working with APIs)
  • LocalStorage & SessionStorage
  • Error handling
  • Mini Projects:
    • Weather App using API
    • Quiz App

Phase 4 (Days 31–40): Version Control + Hosting + GitHub

Goal: Start managing code like a developer

Topics:
  • Git & GitHub basics
  • Push, pull, clone, commit, branch
  • Deploying websites (Netlify / GitHub Pages / Vercel)
  • Mini Project:
    • Host your Portfolio Website

Phase 5 (Days 41–50): React.js (Frontend Framework)

Goal: Learn modern frontend framework

Topics:
  • React basics, components, JSX
  • Props, State, Hooks (useState, useEffect)
  • Conditional rendering, Lists
  • Routing (React Router)
  • Mini Projects:
    • Notes App
    • Movie Search App

Phase 6 (Days 51–60): Backend with Node.js & Express

Goal: Learn to build server & APIs

Topics:
  • Node.js intro, npm
  • Express.js: routes, middleware
  • REST API creation
  • JSON handling
  • Mini Project:
    • Simple API (quotes / products / users)

Phase 7 (Days 61–70): Database (MongoDB)

Goal: Connect backend with database

Topics:
  • MongoDB basics (documents, collections)
  • CRUD operations
  • Mongoose
  • Connecting MongoDB with Express
  • Mini Project:
    • User Authentication System

Phase 8 (Days 71–80): Full Stack Integration (MERN)

Goal: Connect frontend + backend

Topics:
  • React + Node + Express + MongoDB setup
  • Connecting frontend with backend API
  • Authentication (JWT)
  • CRUD full-stack project
  • Mini Project:
    • Blog App / Notes App

Phase 9 (Days 81–90): AI + Full Stack Integration

Goal: Add AI features to your projects

Topics:
  • Introduction to AI & APIs (OpenAI API, Gemini, Hugging Face)
  • Integrate AI chat or image generation
  • Use AI for text summarization or Q&A
  • Mini Projects:
    • AI Study Assistant
    • AI Content Generator

Phase 10 (Days 91–100): Final Project + Portfolio + Resume

Goal: Become job/internship-ready

Tasks:
  • Build & deploy your Final AI Full Stack Project (choose one):
    • AI-based Resume Analyzer
    • Smart Notes App (AI summarizer)
    • Study Assistant Website
  • Create your Portfolio (React or HTML)
  • Add all projects on GitHub
  • Write your Resume + LinkedIn setup
  • Make demo video for your project
