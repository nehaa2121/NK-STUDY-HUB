My First Web Page

Project Idea:

Create a personal introduction webpage that uses basic HTML structure and text formatting tags.

Features to Include

  1. Page title
  2. Your name and short intro
  3. Basic info - like hobbies, goals, favorite quote
  4. Use of formatting tags:
    • <b>, <i>, <u>
    • <small>, <big>
    • <sup>,<sub>
    • <mark>, <strong>, <em>
    • Headings <h1> to <h6>
    • Line breaks <br> and paragraph <p>

Code:

            <!DOCTYPE html>
            <html>
            <head>
                <title>My First Web Page</title>
            </head>
            <body>
                <h1>Welcome to My Web Page!</h1>
                <h2>About Me</h2>
                <p>Hello! My name is <b>Neha</b>. I am <i>Computer Science student</i> learning <u>Web Development</u>.</p>

                <h3>My Goals</h3>
                <p>I want to become a <strong>Full Stack Developer</strong> and create projects that <em>help people</em>.</p>

                <h3>My Goals</h3>
                <p>I want to become a <strong>Full Stack Developer</strong> and create projects that <em>help people</em>.</p>

                <h3>My Hobbies</h3>
                <p>I love <mark>designing, coding, and reading</mark> about technology.</p>

                <h3>My Favourite Quote</h3>
                <p><q>Keep learning , keep growong!</q></p>

                <h3>Some Fun with Text</h3>
                <p>Water formula: H<sub2</sub>0 <br>
                         Square formula: a<sup>2</sup></p>

                <p><small>This is my first mini project using HTML basics and formatting tags.</small></p>
            </body>
            </html>
